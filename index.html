MY PROJECTS

Weather Forecast

Implemented a responsive Weather Forecast Project using HTML, CSS, and JavaScript, designed to provide real-time weather updates for any city. The application not only displays essential weather details like temperature (in degrees Celsius), humidity levels, and wind speed, but it also dynamically adjusts the weather icon based on current climatic conditions, offering a more interactive and visually engaging experience. The project is fully responsive, ensuring optimal performance and user experience across various devices, whether it's a smartphone, tablet, or desktop. This project showcases my ability to integrate front-end technologies effectively, resulting in a functional and user-friendly weather forecasting tool.

BMI Calculator

Developed a responsive web design that dynamically calculates and displays the user's Body Mass Index (BMI) based on the input they provide, such as height and weight. The design not only computes the BMI but also provides a visual indication of where the user's BMI falls within the normality range, helping users easily understand whether they are underweight, normal weight, overweight, or obese. The interface is intuitive and adapts seamlessly across different devices, ensuring a consistent user experience on desktops, tablets, and smartphones.

Color Selector

Created a responsive web project utilizing HTML, CSS, and JavaScript that dynamically changes the background color of the page based on the color the user inputs. The user can easily type in any color name or hex code, and the page will immediately update to reflect that color as the background. Additionally, I implemented a reset functionality, where a dedicated reset button allows the user to revert the background to its original state with a single click. The project is designed to be fully responsive, ensuring optimal performance and user experience across various devices and screen sizes.
